# django-practices
1. 快速实现数据库管理系统
2. 快速实现数据库的Restful接口

```
目前只兼容Python3.x
```

## 第三方框架
* Django
* Django Admin
* Django REST framework 

## 愿景
花最少的时间，快速撸出功能多、Bug少的代码。


## 安装步骤

```
yum install -y git

git clone https://github.com/zz-practices/django-practices.git

cd django-practices/src

./install.sh

./init_db.sh

./startup.sh

iptables -F
```
